About This Portal

This portal provides canonical schema implementation guides for HealthLX client data submissions. Each guide documents the XSD structure, field definitions, cardinality, and practical guidance for CMS compliance and FHIR interoperability.

Maintained by HealthLX. For questions or support, contact your HealthLX implementation team.
